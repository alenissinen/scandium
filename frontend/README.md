## Development setup

### Prerequisites
- Node.js 24+

### Install
```bash
npm install
```

This will automatically install and activate git hooks via lefthook.
Hooks run Biome linting and formatting checks before every commit.

### Commands
| Command | Description |
|---|---|
| `npm run dev` | Start development server |
| `npm run check` | Run Biome linter and formatter |
| `npm run build` | Production build |
